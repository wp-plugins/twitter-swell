====================
描述
====================
添加Twiiter信息输入框到自己的博客上面并个性化输出显示。


====================
安装
====================
1.解压缩ZIP文件到自己BLOG空间的wp-content/plugins下面。
2.进入WP后台激活Twitter Swell插件。
3.在WP后台的Option（操作）管理页面，填写Twitter账户和密码，以及其他可选配置。


====================
使用方法
====================
- 术语约定 -
1. 登录窗体：即登录博客时填写用户名，密码的窗体。
2. Twitter发送窗体：即输入你要发送的Twiiter信息框的窗体。
3. Tweet窗体：即显示Twitter信息的窗体。

在你想显示Twitter发送框的地方插入下列代码到你的主题当中即可：
<?php if (function_exists('WSTS_ShowTwitter')) : ?>
<?php WSTS_ShowTwitter(); ?>
<?php endif; ?>
建议插入到你当前使用主题index.php文件<?php if (have_posts()) : ?> 的上一行。 

令人兴奋的是，WSTS_ShowTwitter()函数还支持参数配置哦！下面是参数的使用：
WSTS_ShowTwitter($LoginForm={true|Fals})
默认使用可以省略两个参数，省略使用表示不显示登录窗体也不显示Tweet窗体。
如果你想在在未登录博客的状态下显示登录窗体。把在模板文件使用WSTS_ShowTwitter($LoginForm=true)代码即可。
使用<?php WSTS_LoginForm() ?>代码也可以显示登录窗体，但它仅仅是显示登录窗体。

在模板文件使用<?php WSTS_TweetForm() ?>代码，这样在插入位置显示Tweet窗体。
如果你不满意当前的样式，你可以在插件目录下的css文件夹下修改相应的设置以满足你的要求。
====================
注意事项
====================
1. 使用本插件显示Tweet窗体的时候，请务必确认没有其他第三方或者官方的显示插件。
